Python

J’ai réalisé ce programme Python durant le lycée, il permet de crée un pierre-feuille-ciseau entre un joueur et un ordinateur sans utilisation d’interface graphihque. C’est-à-dire que tout ce passe dans la console. Le lancement de la partit ce fait avec un appel de la fonction "jeu" qui a besoin en paramètre du nombre de points nécessaire pour gagner. Le programme est divisé en plusieurs fonctions qui une fois assemblé permettent de jouer.

Pour expliquer simplement le fonctionnement, on demande au joueur son choix (Pierre-Feuille-Ciseau), le choix de l’ordi est fait de manèire aléatoire avec un randint. Ensuite, il y a une fonction qui réalise des tests pour déterminer le gagnant. Tant qu’aucun des deux joueur n’a atteint le nombre de point nécessaire pour gagner, le programme continue de lancer des manches. 


from random import randint

points = [0, 0]

def afficher(texte: str):
    print(texte)

def saisir():
    """demande choisir entre 0:pierre, 1:feuille, 2:ciseaux"""
    console = int(input("FAIT TON CHOIX\n 0 pour pierre \n 1 pour feuille \n 2 pour ciseaux \n\n mon choix:"))
    return console

def manche():
    """choix joueur vs choix ordi"""
    
    joueur = saisir()
    ordi = randint(0, 2)
    reponse = (joueur, ordi)
    LesChoix = ("pierre", "feuille", "ciseaux")
    afficher(f"joueur: {LesChoix[joueur]} VS ordi: {LesChoix[ordi]}")
    if reponse[0] == 0 and reponse[1] == 0:
        print("Egalité")
    if reponse[0] == 1 and reponse[1] == 1:
        print("Egalité")
    if reponse[0] == 2 and reponse[1] == 2:
        print("Egalité")
    if reponse[0] == 1 and reponse[1] == 0:
        print("joueur gagne")
        points[0] += 1
    if reponse[0] == 2 and reponse[1] == 0:
        print("ordi gagne")
        points[1] += 1
    if reponse[0] == 0 and reponse[1] == 1:
        print("ordi gagne")
        points[1] += 1
    if reponse[0] == 0 and reponse[1] == 2:
        print("ordi gagne")
        points[1] += 1
    if reponse[0] == 1 and reponse[1] == 2:
        print("joueur gagne")
        points[0] += 1
    if reponse[0] == 2 and reponse[1] == 1:
        print("joueur gagne")
        points[0] += 1
    print(points)
    return points

def jeu(gagne: int):
    """affiche les résultats de manche quand joueur ou ordi gagne"""
    while points[0] < 3 and points[1] < 3:
        manche()
    if points[0] == 3:
        print("\n \n \n JOUEUR GAGNE LA PARTIE")
    if points[1] == 3:
        print("\n \n \n ORDI GAGNE LA PARTIE")

C

Au cours de ma première année de BUT, j'ai conçu ce programme en langage C qui permet de convertir un nombre décimal compris entre 0 et 255.

Pour expliquer de manière concise son fonctionnement, l'utilisateur saisit la valeur à convertir. Le programme procède ensuite à une vérification pour s'assurer que la valeur se situe bien entre 0 et 255. Si c'est le cas, une boucle 'for' est utilisée pour attribuer les valeurs 0 ou 1 à une liste de 8 éléments, chacun représentant un bit de la conversion binaire. Enfin, une boucle 'while' est utilisée pour afficher les valeurs de la liste dans l'ordre inverse, ce qui représente le nombre binaire dans le bon ordre. 


#include stdio.h>
#define T 8

int main() {
    int valeur;
    int bits[T];
    int cpt = 0;
    
    printf("Valeur? ");
    scanf("%d", &valeur);

    if (valeur >= 1 << T) {
        printf("La valeur est trop grande. Veuillez entrer une valeur inférieure à %d\n", 1 << T);
        return 1; 
    }
    printf("La valeur binaire de %d est ", valeur);
    for (int i = 0; i < T; i++) {
        bits[i] = valeur % 2;
        valeur = valeur / 2;
        cpt += 1;
    }

    cpt -= 1;
    while (cpt >= 0) {
        printf("%d", bits[cpt]);
        cpt -= 1;
    }

    return 0;
}
